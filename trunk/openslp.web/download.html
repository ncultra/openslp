Two Lines of Development

The OpenSLP project developers have chosen to maintain two code branches. The Stable branch and the Development branch.

The current stable version is 1.2.1 and has been in active use for several years now by the GNU/Linux community. A few bugs have been found (and fixed) since its release in 2006. We'll release a 1.2.2 version shortly in order to publish these fixes.

The current development version is 2.0 beta 1 and has been in test mode for quite a while. The previous development version was 1.3.x. Since 1.3.x incorporates the entire line of changes for IPv6, it was decided to move these changes into 2.x and make 2.x the new development line.

The OpenSLP project also provides a pure Java implementation of the SLP API. The current version is the first version and has been stable since 2003.
