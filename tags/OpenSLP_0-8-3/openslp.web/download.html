New in 0.8.3

  • Completely updated Win32 port
  • Preliminary support for asyncronous API calls is now a compile time option.  Async calls still not fully supported.
  • Addition of SLPParseAttr() API that complies with what will be in RFC2614bis (Note: the libslp.so library version is now 0.0.3)
  • Many fixes to documentation based on review feedback from debian developer Susan G. Kleinmann (sgk@debian.org).
  • Many bug fixes including the following:
    • "DAs with multiple scopes" bug
    • "Local service time out on DA" bug
    • "Bad 1400 byte message" bug
