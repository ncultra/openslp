SLPClose

Declaration

#include <slp.h>

void SLPClose(SLPHandle hslp)
 

Description

Frees all resources associated with the handle.  If the handle was invalid, the functions returhs silently.  Any outstanding synchronous or asynchronous operations are cancelled so their callback functions will not be called any longer.
 

Parameters

hslp An open SLPHandle that was returned by SLPOpen()

Returns

None
 

Status

OpenSLP 0.6.0 Fully implemented as specified by RFC 2614
