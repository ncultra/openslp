Command Line Options

Currently the OpenSLP daemon (slpd) accepts several useful command line options.
 

Command Line

slpd [-d] [-c config file] [-r registration file] [-l log file] [-p pid file]

Options

[-d]
Don't detach from the controlling tty
[-c config file]
Option to change the file the slpd uses to read configuration information. Defaults to /etc/slp.conf.  See Modifying the slp.conf configuration file for details
[-r registration file]
Option to change the file that slpd reads to obtain static registrations.  Defaults to /etc/slp.conf. See Creating a registration file for details.
[-l log file]
Option to change the file that receives slpd log messages.  Defaults to /var/log/slpd.log
[-p pid file]
Option to change the file that holds the slpd process id.  Defaults to /var/run/slpd.pid
